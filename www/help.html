Getting started

Input/Feature Selection tab

Load the training data on the Input Selection box and select the dependent and independent variables on the Feature selection box.

Model tab

Click the Run button to build the ML models on the Modelling box. The model description, summary and statistics of the training step will be shown on the Results box.

Predict tab

Load the test data on the Input Selection box and Click the Run button on the Predict box to make predictions. The results from each model will be shown on the Predictions box
